# investigate-high-memory

Use when Fly machines show sustained memory pressure.

## Checks

1. Confirm affected app/machine set and time window.
2. Compare current memory vs normal baseline.
3. Check restarts/OOM signals near spikes.
4. Check recent deploy/config change correlation.
